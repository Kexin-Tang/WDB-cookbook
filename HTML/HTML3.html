Concepts

<form>

The form element represents a document section containing interactive controls for submitting information.
The action attribute specifies WHERE the form data should be sent.
The method attribute specifies WHICH http method should be used.

<input>

The input element is used to create a variety of different form controls.
The type attribute is where the magic happens. Changing type alters the input's behavior and appearance.

<label>

Use label to link some text with buttons.

<button>

Set buttton's type to achieve different functions through the <form>'s action attribute.
